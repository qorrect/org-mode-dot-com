Wemacs Help

Wemacs is a keyboard driven editor that is built on functions ( type `M-x TAB` to see a list of all functions available ) . These functions can be mapped to keyboard bindings to execute them quickly.

Notation:

C = Control
M = Alt
S = Shift
<key> = A number, letter, or symbol

Example: C-M-f = Control + Alt + f, all pressed at the same time
Example: C-x C-s = Control + x followed by Control + s

The Essentials:

C-x C-c = close_wemacs
C-g     = cancel
M-x     = execute_function
C-x C-f = open_file
C-x k   = close_file

File Functions:

C-x C-f = open_file
C-x C-f = new_file
C-x C-s = save_file
C-x C-q = safe_file_as
C-x k   = close_file

Navigation Functions:

C-f   = forward_char
C-b   = backward_char
M-f   = forward_word
M-b   = backward_word
C-a   = beginning_of_line
C-e   = end_of_line
C-M-a = beginning_of_buffer
C-M-e = end_of_buffer
C-ARROW_LEFT = backward_word
C-ARROW_RIGHT = forward_word
C-ARROW_UP = backward_paragraph
C-ARROW_DOWN = forward_paragraph

Editing Functions:

TAB    = indent
C-q    = delete_region
C-BACKSPACE = backward_kill_word
M-BACKSPACE = backward_kill_word_no_copy
M-d = kill_word
C-k = kill_line
S-BACKSPACE = backward_delete_whitespace
S-DELETE = delete_whitespace
C-M-d = delete_region_or_line

Selection Functions:

C-SPACE = set_mark_command
C-c c = copy
C-c v = paste
C-c b = paste_second_to_last
C-c n = paste_nth_copy
