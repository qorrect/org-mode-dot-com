Wemacs Help

Wemacs is a keyboard driven editor that is built on functions ( type `M-x TAB` to see a list of all functions available ) . These functions can be mapped to keyboard bindings to execute them quickly.

Notation:

C = Control
M = Alt
S = Shift
<key> = A number, letter, or symbol

Example: C-x C-s --> Control + x followed by Control + s
Example: C-SPACE --> Control + Spacebar
Exception: C-n M-x function_name --> Here n refers to a digit , for example typing Ctrl + 20 then M-x comment_line will execute comment_line 20 times ( to comment 20 lines of code )

File Functions:
(Dark gray is the emacs command)

C-x C-f = open_file (find_file)
C-x C-s = save_file (save_buffer)
C-x C-q = safe_file_as (write_file)
C-x k   = close_file (kill_buffer)

Navigation Functions:
(Dark gray is the emacs command)

C-f = forward_char(forward_char)
C-b = backward_char (backward_char)
M-f = forward_word(forward_word)
M-b = backward_word (backward_word)
C-a = beginning_of_line (beginning_of_line)
C-e = end_of_line (end_of_line)
C-M-a = beginning_of_buffer (beginning_of_buffer)
C-M-e = end_of_buffer (end_of_buffer)
C-ARROW_DOWN = forward_paragraph (forward_paragraph)
C-ARROW_UP = backward_paragraph (backward_paragraph)

Editing Functions:
(Dark gray is the emacs command)

C-SPACE = set_mark_command(set_mark_command)
C-SPACE = set_mark_command(set_mark_command)
