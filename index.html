Org-Mode editing on the web

Edit your org files in the browser using a familiar interface.

Login with Dropbox
 
This app works in a sandbox directory reading files from /Dropbox/Applications/org-mode-dot-com (after you grant it permission, which can be revoked here.) You will need to copy or symlink files to that directory to see them in the app, then you can find them under:

File->Dropbox->(files) or C-x C-f <TAB>

In its current state its only saving files to localStorage not to Dropbox.
